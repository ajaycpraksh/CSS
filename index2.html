The CSS margin property allows us to create space around an element. They define the gap between an element and its neighboring elements. Margins can be set individually for each side (top, right, bottom, left). The CSS padding properties are used to generate space around an element's content, inside of any defined borders.


An element with position: relative; is positioned relative to its normal position.

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).
